Hello! My name is Ricky Wallace, and this is my submission to The Challenge, as requested by Fullstory. This was a fun and indeed challenging project!

For my challenge, I chose to make a very simple web app for one of my hobbies - flying! Before taking any flight, you must calculate weight and balance, and determine how much fuel you can take. Many apps on the market will calculate weight and balance, but fuel is always a guess-and-check operation. So this app will give you an exact fuel amount based on the loading of the aircraft.

To use, simply navigate to the Fuel Calculation page, select an aircraft, and enter the weights. Fuel tolerances will automatically be calculated as you change the values. If you are overweight, the app will tell you how much weight you need to cut, and if you are within 15 gallons, the text will be orange to indicate the low fuel availability.

You can add or remove aircraft on the Aircraft page using the buttons available.

I decided to further challenge myself in this project by using:

  • Firebase, the free Google noSQL database-as-a-service toolset. I had not used it prior, but found it fairly simple to implement.
  • Chart.js for rendering the charts on the GitHub page. I found it to be a pretty powerful tool for simple charts.
  • In addition, the CSS is a modified version of a template from W3 that includes responsive elements and some modern CSS layouts.

Disclaimers:

  • This was written from scratch in a few days. Please excuse the lack of...........coding finesse :)
  • This website is a demonstration. Do not use this to actually calculate Weight and Balance for preflight. There are other factors that go into Weight and Balance that should be considered.
